## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to a project folder:

```bash
cd Level_1_Basic
```

3. Run a script:

```bash
python filename.py
```

For Django projects, ensure dependencies are installed and run:

```bash
python manage.py runserver
```
